# TBW AI PREMIUM NAVIGATOR (demo bundle)

- React + Vite front-end
- PWA manifest + service worker
- Fixed header, ticker, hero, cards
- Trial / Demo / Premium UI modes
- Multilingual texts (24 jezika, jezici u `translations.js`)
- Weather widget u headeru (demo preset po gradovima)
- Modal prozori za navigaciju, smještaj, vrijeme, promet, evente, trgovine i SOS

## Pokretanje lokalno

```bash
npm install
npm run dev
```

Za produkciju:

```bash
npm run build
```

Deployaj `dist/` na Netlify ili Vercel (Static Site).

### Napomena

Ovaj bundle je front-end dio. TBW NAVENGINE (AI, sigurnosne funkcije, plaćeni API-jevi) trebaš spojiti na svoj backend / serverless funkcije.
